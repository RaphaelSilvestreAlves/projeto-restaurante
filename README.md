# Projeto Restaurante

API back-end para cadastro de restaurantes, desenvolvida em Python.  
O projeto simula uma funcionalidade de um aplicativo de entrega de comida, permitindo cadastrar, listar, atualizar e remover restaurantes.

## Objetivo

Este projeto foi criado para praticar conceitos de desenvolvimento back-end, como:

- criação de APIs REST
- operações CRUD
- organização de rotas
- validação de dados
- integração com banco de dados

## Funcionalidades

- Cadastrar restaurante
- Listar restaurantes
- Alternar estado do restaurante

## Tecnologias utilizadas

- Python
